This guide walks you through validating a KSeF FA(3) XML invoice using **ksefuj.to**.

## Step 1: Open the validator

Go to [ksefuj.to](https://ksefuj.to) — a free, privacy-first XML validator for KSeF.

## Step 2: Upload your XML file

Drag and drop your XML file into the validation zone, or click to select it.

<Info>
All processing happens **in your browser**. No file is ever sent to a server.
</Info>

## Step 3: Review the results

The validator will show:

- **Errors** — critical issues that must be fixed before submitting to KSeF
- **Warnings** — potential issues worth reviewing
