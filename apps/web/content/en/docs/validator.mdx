The **ksefuj** validator checks XML files for compliance with the FA(3) schema from the Polish
Ministry of Finance. It works in two stages:

1. **XSD Validation** — checks the XML structure against the official schema
2. **Semantic Validation** — checks business rules that XSD cannot express

## Usage

### Browser (online)

Go to [ksefuj.to](https://ksefuj.to) and drag your XML file into the validation zone.

### CLI

```bash
npx @ksefuj/validator invoice.xml
```

### npm package

```typescript
import { validate } from "@ksefuj/validator";

const result = await validate(xmlString);
console.log(result.errors);
console.log(result.warnings);
```
