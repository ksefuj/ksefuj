Цей посібник пояснює, як перевірити файл XML рахунку-фактури KSeF за допомогою **ksefuj.to**.

## Крок 1: Відкрийте валідатор

Перейдіть на [ksefuj.to](https://ksefuj.to) — безкоштовний онлайн-валідатор XML для KSeF.

## Крок 2: Завантажте файл XML

Перетягніть файл XML у зону валідації або натисніть, щоб вибрати файл.

<Info>Усі дані обробляються **лише у вашому браузері**. Жоден файл не надсилається на сервер.</Info>

## Крок 3: Перегляньте результати

Валідатор покаже:

- **Помилки** — критичні проблеми, які треба виправити перед відправкою до KSeF
- **Попередження** — потенційні проблеми, які варто перевірити
