## Виставлення рахунків

### Як виставити перший рахунок у KSeF?

Спочатку отримайте доступ до KSeF — увійдіть через Aplikację Podatnika KSeF (ap.ksef.mf.gov.pl) або інтегруйте свою програму виставлення рахунків з API KSeF. Потім підготуйте XML-файл відповідно до FA(3) і надішліть його до системи. Після успішного прийняття отримаєте номер KSeF та UPO.

### Як виглядає коригувальний рахунок у KSeF?

Коригувальний рахунок у KSeF повинен містити номер KSeF оригінального рахунку — без цього поля коригування буде відхилено. Ви надсилаєте його так само, як звичайний рахунок: як XML-файл до KSeF. Після прийняття він отримує власний номер KSeF.
